# Build Setup

### install dependencies
```bash
$ npm install
```

### serve with hot reload at [localhost:3000](http://localhost:3000/)
```bash
$ npm run dev
```
### build for production and launch server
```bash
$ npm run build
$ npm run start
```
### generate static project
```bash
$ npm run generate
```
